-===========
 오브젝트 분류
============

1. Block
----------------------------------------------
 1) Block
 - 일반 장애물

 2) barbed block (가시달린 블록)
 - 움직임, 플레이어의 체력을 깎음
----------------------------------------------


2. Button 
----------------------------------------------
 1) Red Button  ->EventScript 추가
 - Stone Door 과 Event
 - 밟으면 문이; 열리거나 특정 조건 수행

 2) Speed Up Board
 - 밟으면 앞으로 이동 
 3) 점프 발판(아직 추가 안함)
 - 밟으면 점프해서 맵 이동  
----------------------------------------------


3. Etc
----------------------------------------------
 1) Bomb, BombFlower (폭탄, 폭탄꽃)
 - 꽃에서 폭탄 생성
 - 폭탄을 던질 수 있고 breakable stone 부술 수 있음

 2) FireWood , FireBowl (화로, 장작) ->EventScript 추가
 - 화로에는 불이 붙어있음
 - 화로를 들고 장작에 가까이 가면 불이 붙음
 - 불이 붙으면 문이 열리거나 특정 조건 수행
----------------------------------------------


4. Stone
----------------------------------------------
 1) BreakableStone, BreakableSmallStone
 - 플레이어가 부술 수 있음, 폭탄으로도 가능

 2) PushStone , PushSmallStone
 - 플레이어가 밀 수 있음 , 밀어서 버튼 누를 수 있음

 3) StoneDoor
 - 특정 조건 수행 시, 문이 열림
----------------------------------------------
